# 🛒 Amazon SQL Data Analysis Project

## 📌 Project Overview
This project analyzes Amazon product data using SQL to extract insights about:
- Product performance
- Customer engagement
- Pricing strategy
- Reviews and ratings

## 🧠 Key Insights
- Identified top revenue-generating products
- Analyzed category performance
- Found hidden gem products (high rating, low reviews)
- Studied relationship between discount and ratings

## 🛠️ Tools Used
- MySQL
- SQL (Joins, Aggregations, Window Functions)

## 📂 Files
- queries.sql → Contains all analysis queries
